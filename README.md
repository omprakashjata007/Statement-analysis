# Statement Analysis Project

## Overview
This project performs statement analysis on customer reviews using Python and TextBlob.

## Objectives
- Analyze customer reviews
- Classify Positive, Negative, and Neutral statement
- Visualize sentiment distribution
- Export results to Excel

## Tools Used
- Python
- Google Colab
- Pandas
- TextBlob
- Matplotlib
- Microsoft Excel

## Project Files
- reviews.xlsx
- Statement_analysis.py
- Statement_Result.xlsx
- statement analysis.pdf
- Output screenshots

## Output
- Statement classification
- Bar chart
- Excel output
